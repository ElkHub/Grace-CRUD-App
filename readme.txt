Setup Instructions:
To run the project locally, follow these steps:

Clone the repository to your local machine.
Import the mypcot_db.sql database file into your MySQL server.
Update the database connection settings in the project files to match your MySQL credentials.
Install project dependencies 
Launch your web server and visit the project in your browser.
